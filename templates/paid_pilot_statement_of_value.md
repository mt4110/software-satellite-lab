# Paid Pilot Statement of Value

## Offer

Local-first AI-assisted development evidence audit for one repo or workflow.

## Included

- Setup and no-upload review.
- Passive capture configuration.
- Signed evidence report.
- Transcript claim audit.
- Failure-memory extraction workshop.
- Optional sanitized failure signature starter pack.
- Final adoption dossier.

## Not Included

- Code generation.
- Production security certification.
- Automatic cloud upload.
- Guaranteed vulnerability detection.

## Success Criteria

- First audit report generated.
- At least one real repeated-failure pattern captured.
- No private source uploaded.
- Team can verify the report offline.
- Buyer identifies the next adoption step.
