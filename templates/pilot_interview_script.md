# Pilot Interview Script

## Opening

We are testing a local-first evidence layer for AI-assisted software development. It does not write code. It records what happened, verifies artifacts, and helps prevent repeated failures.

## Questions

1. Which AI coding tools do you use today?
2. What happens when the tool says "tests passed"?
3. Do you keep durable evidence of AI-assisted work?
4. What failures repeat across projects?
5. Are you allowed to upload source code, logs, or transcripts to a third-party SaaS?
6. Would a signed local audit report help with PR review, client work, or governance?
7. Would team failure-memory sharing help?
8. What would make this unacceptable?
9. What would you pay for as a pilot?
10. Who else would need to approve?
